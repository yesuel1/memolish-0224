# 프로젝트 메몰리쉬 (Memolish) MVP PRD
**작성일:** 2026-02-23
**프로젝트 목표:** 일상 메모와 할 일(To-do)을 초개인화된 영어 학습 대화문으로 변환하는 하이브리드 웹 서비스 구축

## 1. 서비스 개요 및 타겟
* **핵심 가치:** "내 삶이 곧 영어 교과서가 된다." 기존의 뻔한 예문이 아닌, 내가 방금 적은 진짜 고민과 스케줄로 영어를 학습함.
* **타겟 고객:** 실용적인 영어 회화에 목마른 직장인, 대학생 (초기 숏폼 마케팅을 통해 유입 예정).

## 2. 핵심 유저 동선 (Two-Track UI/UX)
* **Track 1: 생산성 모드 (기본 화면)**
    * 텍스트, 사진, 유튜브/웹 링크 입력 기능.
    * 입력 시 시작일(현재) ~ 종료일(내일) 자동 할당.
    * 칸반 보드 또는 리스트 형태로 상태 관리 (진행 전, 진행 중, 완료, 계속 참조).
* **Track 2: 학습 모드 (AI 변환)**
    * **수동 변환 원칙:** 유저가 특정 메모에서 `[✨ 영어로 변환하기]` 버튼을 클릭했을 때만 AI 동작 (API 과금 방어).
    * 변환 시, 원본 메모의 내용을 한/영으로 요약.
    * 해당 주제를 바탕으로 바로 실생활에 쓸 수 있는 A/B 롤플레이 영어 대화문 생성.
    * **음성 다운로드:** 생성된 대화문을 TTS API를 거쳐 오디오 파일로 다운로드 가능하게 구현.

## 3. 기술 스택 (권장)
* **Frontend:** React.js 기반 웹 (모바일 최적화 필수)
* **Backend:** Node.js 또는 Python FastAPI
* **AI & Voice:** Google AI Studio (Gemini API - 텍스트/링크 요약 및 스크립트 생성), Google Cloud TTS API
* **Database & Storage:** PostgreSQL (데이터 관리), AWS S3 (음성 파일 임시 생성 및 다운로드)

## 4. 비즈니스 모델 (성장형 하이브리드)
* MVP 개발 시 회원가입 및 크레딧(Credit) 시스템 기반 마련 필수.
* **무료 유저:** 생산성 기능 평생 무료. 매일 밤 12시 'AI 변환 크레딧' 3개 리셋.
* **보상형 광고:** 크레딧 소진 후 변환 시 15~30초 동영상 광고 시청.
* **프리미엄 (월구독):** 무제한 변환, 무제한 음성 다운로드 등.

## 3. 컴포넌트 명세서

### 3-1. `<CreditBadge />`

| 항목 | 내용 |
|------|------|
| **위치** | 헤더 우측 |
| **표시** | `✨ {n}/3` |
| **상태** | n > 0: 초록 / n = 0: 빨강 / 프리미엄: 💎 |
| **Props** | `credits: number`, `isPremium: boolean` |
| **클릭** | 크레딧 설명 툴팁 표시 |

### 3-2. `<MemoInputPanel />`

| 항목 | 내용 |
|------|------|
| **트리거** | FAB(+) 클릭 → 바텀 시트 슬라이드 업 |
| **닫기** | 배경 탭 또는 아래로 스와이프 |
| **필드** | `content` (필수, textarea), `source_url` (선택) |
| **동작** | URL 입력 시 자동 메타데이터 파싱 → 미리보기 표시 |
| **날짜** | start=오늘, end=내일 자동 세팅 (수정 가능) |
| **제출** | `POST /api/memos` 후 보드 업데이트 |

### 3-3. `<MemoBoard />`

| 항목 | 내용 |
|------|------|
| **레이아웃** | 리스트 뷰 (MVP) / 칸반은 v2 |
| **탭 필터** | 전체 / 진행전 / 진행중 / 완료 / 계속참조 |
| **정렬** | 최신순 (created_at DESC) |
| **빈 상태** | "첫 메모를 작성해 보세요!" 안내 문구 |

### 3-4. `<MemoCard />`

| 항목 | 내용 |
|------|------|
| **상태 변경** | 카드 내 상태 뱃지 탭 → 드롭다운 선택 |
| **URL 표시** | 링크 있을 때 파비콘 + 제목 미리보기 |
| **핵심 CTA** | `[✨ 영어로 변환하기]` — 미변환 시 표시 |
| **변환 완료** | `[✅ 대화문 보기]` — 변환 완료 시 표시 |
| **삭제** | 스와이프 투 딜리트 또는 롱프레스 메뉴 |

### 3-5. `<LearningModal />`

| 항목 | 내용 |
|------|------|
| **트리거** | `[✨ 영어로 변환하기]` 클릭 → `POST /api/ai/transform/{id}` 호출 |
| **로딩** | 스켈레톤 + "✨ AI가 대화문을 만들고 있어요..." |
| **크레딧 0** | 광고/프리미엄 업셀 UI로 전환 |
| **콘텐츠** | 원본 메모 → 한/영 요약 → A-B 대화문 |
| **TTS** | `[🔊 듣기]` → `POST /api/audio/generate/{id}` |
| **다운로드** | `[⬇ 다운로드]` → `GET /api/audio/download/{id}` (presigned URL) |
| **스피커 색상** | A: blue-500 / B: orange-500 |

### 3-6. 상태 관리 (Zustand Store 구조)

```typescript
interface MemoStore {
  // 데이터
  memos: Memo[]
  credits: { daily: number; isPremium: boolean }

  // 메모 CRUD
  fetchMemos: () => Promise<void>
  createMemo: (data: CreateMemoInput) => Promise<void>
  updateStatus: (id: number, status: MemoStatus) => Promise<void>
  deleteMemo: (id: number) => Promise<void>

  // AI 변환 (수동 트리거만)
  transformMemo: (id: number) => Promise<TransformResult>

  // 오디오
  generateAudio: (id: number) => Promise<string>   // → audio URL
  getDownloadUrl: (id: number) => Promise<string>  // → presigned URL

  // UI 상태
  activeFilter: MemoStatus | 'all'
  setActiveFilter: (filter: MemoStatus | 'all') => void
  inputPanelOpen: boolean
  setInputPanelOpen: (open: boolean) => void
  learningModalMemoId: number | null
  setLearningModalMemoId: (id: number | null) => void
}
```
